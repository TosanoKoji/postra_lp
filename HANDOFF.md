## 1. プロジェクト概要

**Postra** — Mac 向けの写真セレクト／組写真／簡易レタッチアプリのランディングページ。

- **プロダクト性格**: 「写真再発見」がキーメッセージ。ストレージに眠る数万枚から「あの1枚」を見つけるためのアプリ。
- **価格**: 買い切り ¥5,800（Paddle 決済想定）
- **対象**: Mac（Apple Silicon / Intel、macOS 12 以降）。Windows 版は後日対応予定。
- **競合関係**: Lightroom / Capture One とは**競合せず併用を想定**。Postra は JPG 撮って出し or 現像済みの管理・選別・組写真・簡易編集・書き出しに特化。RAW は埋め込み JPEG プレビューで閲覧可（現像はしない）。

## 3. 技術スタック / デザイン方針

- **形式**: 単一 HTML ファイル。フレームワーク不使用。CSS は `<style>` にインライン。JS は最小限（reveal アニメーションと details の制御程度）。
- **カラーパレット**: ダークモード基調。`:root` に CSS 変数で定義されています。
  - 背景: `#0a0b0d` 系
  - カード: `#12141a` 系
  - アクセント（オレンジ寄り）: `var(--accent)`
- **フォント**: システムフォント（日本語はヒラギノ角ゴ / Noto Sans JP）+ `ui-monospace` を小さなラベル用に。
- **セクション構造**: `<section data-screen-label="XX-name">` で番号ラベルを付与。並び順の指標になります。

### 🔴 最優先: 実 UI の動画／GIF への差し替え

現状、§3〜§6 のビジュアルはすべて **CSS / HTML で作った汎用的なモック**です。実際の Postra の UI とは見た目が異なるため、**このまま公開するとユーザーに誤解を与えます**。

ユーザー（プロダクトオーナー）が各セクションのアプリ操作録画を用意します。取り込み時は `<video autoplay loop muted playsinline>` で差し込みます（GIF より圧倒的に軽くて綺麗）。

**動画差し込みの優先順位**（ユーザー合意済み）:

1. **§5 レタッチ**（`data-screen-label="05-retouch"`）← ★ 最優先。現状の Before/After スライダーは**実 UI と別物**なので「嘘」になってしまう。ユーザー本人も一番気にしていた箇所。
2. **§4 デュアル A/B**（`04-compose`）
3. **§6 爆速スクロール**（`06-speed`）
4. **§3 AI 類似検索**（`03-ai`）
5. **§1 ヒーロー**（`01-hero`）← 余裕があれば

**推奨動画仕様**（各セクション、ユーザーに伝え済み）:

| セクション | サイズ | 秒数 | 縦横比 |

## 6. 編集時の注意点・お作法

### 既存のスタイルに合わせる
- 新しい色は勝手に増やさない。`:root` の CSS 変数を使う。
- フォントウェイトは `400 / 600 / 700` の 3 段階のみ。それ以外は基本使わない。
- セクション間の余白は既存のパターンに揃える（だいたい `padding: 120px 0` くらい）。

### コピーの書き方
- 体言止め・句点ありを基本に。
- 技術用語（JPEG / RAW / macOS / Apple Silicon 等）は**半角英数**で統一。
- 「Postra は〜」で始まる文を多用（主語を明確に）。

### 絶対に避けること
- **トライアル動線を復活させない**（ユーザーは既にこの方針で確定済み）
- **実 UI と異なるビジュアルで「嘘」にならないよう注意**（§5 レタッチはまさにこの理由で動画差し替え予定）
- **絵文字を入れない**（ブランドトーンに合わない）
