01 · 写真を「探す」楽しさ、再び見つけませんか？

ストレージに眠る数万枚。
「あの1枚」を、見つけに行く。

「いつか見返そう」で溜まり続ける写真フォルダ。せっかく撮った瞬間が、ただの重いデータになっていく。 Postra は、そこに楽しみを見つけるためのアプリです。

Before

フォルダを眺めるだけで、夕方になる。

撮影日ごとに放り込まれた無名の JPEG。頑張ってRAWから現像したものの、たまり続ける写真たち。 プレビューは遅い、ブラケットで撮った似たカットが並んで、どれが本命か思い出せない。気づけば選ぶ前に集中力が切れる。

IMG_0001 – IMG_9827.jpg
After

選ばれた一枚（組み写真でも）を、すぐ手元に。

似た写真は自動でまとめ、キーボードだけで採用・不採用。右パネルに移せば、もう組写真の編集画面です。「選ぶ（選んでもらう） → 組む → 編集する →書き出す」が、ひとつのアプリで簡潔します。

02 · 類似検索 + キーワード検索

"こういう感じ" を、
言葉にしなくても。

1枚を選び、クリックすれば、意味が近い写真と見た目が近い写真が瞬時に並びます。
「夕焼け」「笑顔」と打てば、フォルダ全体から引き上げます。
タグ付け不要。初期設定をしてしまえば、次からは写真を読み込むたびに自動で付与されます。

「似ている」 の軸を、使い分ける。

同じ被写体を追いかけた連写は 見た目が近い。
別の日に似た構図で撮った一枚は 意味が近い。
Postra はその二つを別のモデルで見て、瞬時に切り替えられます。

MEANING
意味で類似画像を探す（CLIP）「夕暮れ時の海」「横断歩道を渡る人」「雨の夜」のような、状況で一致する写真を選別。
VISUAL
見た目で類似画像を探す（DINOv2）見た目が似ている、連続カットのベストショット探し、同じ色調の写真をまとめる、といった用途に。知らなかった自分の「クセ」もみつかるかも？
KEYWORD
キーワードで探す自然言語で打てば、Exif にタグを付けていなくても検索可能。
「逆光」「赤い傘」もそのまま。
RANDOM
ランダムで発見するライブラリからシャッフルして組み写真に配置。自分でも忘れていた組み合わせを掘り起こす。「頑張って選ぶ」じゃなくて「選ばれる」側に回る時間も、たまには良いかも。

03 · デュアル A/B レイアウト

3枚、4枚、自由なレイアウト。
迷ったら、両方組んでみる。

右パネルは2つに分割も可能、同じ素材または別の素材で別レイアウトを並行して組むことも。並べて見比べて、残す方を書き出す。
SNS 用の 1:1 / 4:5 / 16:9 / 9:16 すべてに対応。変更もプリセット保存も自由に。

A · 2枚組（縦2）1:1
B · 3枚組（メイン＋サブ2）4:5
1:1 4:5 9:16 16:9 3:4 2:3 自由

ドラッグ&ドロップで差し替え。⌘1〜⌘4 でモード切替。Z でランダム入れ替え。

05 · 爆速

10,000枚でも、止まらない。

初回スキャンは少し時間がかかります。そこから先は、スクロールもレイアウトも書き出しも、指の動きに遅れません。JPGに特化したアプリだからこそ実現できる応答速度です。

10,000枚のフォルダを、15分で読み込み。

一度読み込めば以降は即応答。追加でフォルダに入れた写真は「スキャン」ボタンで差分だけ取り込みます。フォルダをピン留めしておけば、次に開くときほとんど待たせません。

よくある質問

特化したアプリです。ご注意ください。

買い切りとサブスク、どちらですか？

買い切りです。一度購入したバージョンは、アップデートが止まってもそのまま使い続けられます。メジャーアップデートが今後出た場合は（例: 2.0）には優待価格で移行できます。

Windows 版はありますか？

現在は Mac 版を優先リリースしていますが、Windows 版も並行して開発中です（近日対応予定）。同じ機能セットを目標に開発しています。

初回起動時に「開発元を確認できません」や「PC が保護されました」と警告が出ます。

OS 標準のセキュリティ機構による警告で、Postra のインストーラに コードサイニング証明書 が付与されていないために表示されます。Postra そのものに問題があるわけではありません（証明書は年額の運用コストがかかるため、初期リリース段階では取得を見送っています。販売実績に応じて将来的に導入予定です）。

Mac の場合（Gatekeeper）: ダウンロードした DMG / アプリを 右クリック → 「開く」 → 確認ダイアログで再度 「開く」を選択。

Windows の場合（SmartScreen）: 警告画面の 「詳細情報」をクリック → 「実行」。

いずれも初回起動時のみで、一度許可すれば以降は警告なしで起動できます。

RAW の現像はどの程度できますか？

RAW現像は全く想定していません。
Postra は RAW埋め込みJPEG プレビューで軽快に表示可能（画質はカメラ依存）です。
現像作業は Lightroom / Capture One / RawTherapee など、お好みのソフトに渡してください。現像したものを取り込むのがベストな運用です。

Lightroom や Capture One と併用できますか？

できます。
むしろ併用を想定して作られています。Postra はJPG 撮って出し、もしくは現像された写真の管理・選別と組写真、簡易編集、各種書き出しに特化したソフトです。
RAWを取り込んだ場合、簡易プレビューで採用した写真だけを現像ソフトに渡す操作も備えています（レイアウト機能、AI検索機能は利用可能です）
ただし、RAWの埋め込みjpg画像は実用に耐えうる解像度でないものも多いです。

写真はクラウドに送られますか？

送られません。類似検索・キーワード検索・レタッチ、他すべての機能が端末のローカルで完結します。
ネットワーク接続は基本的に不要です（AI機能の初期導入やアップデート確認を除く）。

バグや不具合には対応しますが、新機能などについては「思いつき」次第です。
